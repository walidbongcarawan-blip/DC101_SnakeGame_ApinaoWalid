# DC101_SnakeGame_ApinaoWalid
A mobile friendly Snake Game with sound, settings, and animations with theme changes and Snake Skin
## Technologies Used
- HTML
- CSS
- JavaScript
- GitHub
- ## Features
- - Increasing speed difficulty
- Sound effects
- High score using localStorage
- Mobile touch controls
- Theme switcher
- Responsive design
- ## How to Run
1. Download or clone repository
2. Open index.html in browser
3. Use arrow keys or on-screen buttons
